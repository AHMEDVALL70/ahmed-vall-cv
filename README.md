# Iris Flower Classification - Machine Learning Project

## 📝 Project Overview
This project is a machine learning application built with **Python** to classify three different species of Iris flowers (Setosa, Versicolor, and Virginica) based on their physical measurements (sepal length, sepal width, petal length, and petal width).

## 🚀 Key Features
* **Algorithm:** K-Nearest Neighbors (KNN).
* **Library:** Scikit-learn, Pandas, NumPy.
* **Accuracy:** Achieved **100% accuracy** on the test dataset.
* **Environment:** Developed and tested on Google Colab.

## 🛠️ How it Works
1. **Data Loading:** Using the famous Scikit-learn Iris dataset.
2. **Preprocessing:** Splitting data into 70% training and 30% testing.
3. **Training:** Training the KNN model with `n_neighbors=3`.
4. **Prediction:** The model takes new measurements and predicts the correct species.

## 📊 Results
The model successfully identifies flower species with high precision, making it a robust example of supervised learning.
